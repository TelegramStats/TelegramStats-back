#

### The backend for TelegramStats website

#

### installation

```
$ git clone https://github.com/telegramstats/telegramstats-back.git
$ cd telegramstats-back
```

#

### install yarn

```
$ npm install yarn
```

#

### install dependencies

```
$ yarn install
```

#

### development

```
$ yarn dev
```

#

### run server

```
$ yarn server
```
